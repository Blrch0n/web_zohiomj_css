Хичээлийн тайлбар

Программчлалын үндсэн ойлголт, алгоритм бичих, өгөгдлийн бүтэц, объект хандалтат программчлал зэрэг чухал сэдвүүдийг Python хэл дээр судална. Практик төсөл хийх замаар код бичих чадвар эзэмшинэ.
